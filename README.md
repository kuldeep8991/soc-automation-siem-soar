# SOC Automation using SIEM & SOAR

## 📌 Project Overview
This project demonstrates Security Operations Center (SOC) automation using SIEM and SOAR tools to detect and respond to security incidents.

## 🛠 Tools Used
- SIEM: Splunk / Wazuh / Microsoft Sentinel
- SOAR: Slack API
- Scripting: Python

## ⚙️ Workflow
1. Logs collected from system
2. SIEM detects suspicious activity
3. Alert generated
4. SOAR triggers automation
5. Incident response executed
